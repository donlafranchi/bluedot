# Frontend (Next.js)

## Setup
```bash
cd frontend
npm install
npm run dev
```

## Pages
- `/` – Story feed
- `/stories/[id]` – Story detail with emotion prompt, explanation, chart, actions, feedback

## Styling
TailwindCSS configured via `tailwind.config.ts`. Global styles in `app/globals.css`.

## Testing
- `npm run lint`
- `npm run lint:types`
